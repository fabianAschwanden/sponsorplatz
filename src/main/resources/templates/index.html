Wo Vereine und die CSS
zueinander finden.

Eine Plattform. Vier Akteure. Eine Mission. Die kuratierte Anlaufstelle für Schweizer Sport- und Health-Vereine und das CSS Sponsoring-Team — strikt fokussiert, breit gefasst.

Marktplatz entdecken Verein registrieren
Verein. Schweizer Sport- und Health-Vereine bewerben sich mit strukturiertem Profil und werden von der CSS gefunden — strikt fokussiert auf Sport und Gesundheit.
CSS. Das zentrale Sponsoring-Team kuratiert die Plattform, prüft Anfragen einheitlich und macht das CSS-Engagement sichtbar.
Agentur. Regionale CSS-Vertretungen arbeiten im gemeinsamen Workflow mit dem Zentrum — keine Doppelspuren, keine Datenverluste.
Versicherte. CSS-Versicherte sehen, welche Vereine in ihrer Region durch die CSS unterstützt werden — Engagement landet, statt zu verpuffen.
Unser Fokus: Sport Bewegung Reha Mental Health Ernährung Prävention Selbsthilfe

Phase 0 — In Aufbau.

Der CSS Sponsoring-Hub ist die kuratierte Plattform für Sport- und Health-Sponsoring der CSS. Wir nehmen ausschliesslich Vereine auf, deren Mission klar im Sport- oder Gesundheitsbereich verortet ist — Bewegung, Reha, Mental Health, Ernährung, Selbsthilfe. So wird das CSS-Sponsoring zum verlässlichen Treffpunkt für Vereine, das Sponsoring-Team, die regionalen Agenturen und die Versicherten.
