2-Faktor-Authentifizierung einrichten

Zweiter Faktor (TOTP) per Authenticator-App schützt dein Konto auch dann, wenn dein Passwort kompromittiert wird.

1. App installieren

Eine Authenticator-App generiert alle 30 Sekunden einen neuen Code. Empfehlung: 1Password, Microsoft Authenticator, Authy oder Google Authenticator.

2. QR-Code scannen

Falls dein Authenticator keinen QR-Scan kann, gib stattdessen den folgenden Code manuell ein:

ABCDEF...

Algorithmus: SHA1 · 6 Stellen · 30s Periode (RFC 6238 Standard)
